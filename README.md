# 📣 Project

### 🗂 프로젝트 설명

<pre>“실시간 여행계획플랜 서비스! ”

너나들이는, 함께 여행을 가고싶은 사람들과 
실시간으로 계획을 작성하고 공유할 수 있는 서비스입니다 :)</pre>

- 개발 언어 : Javascript
- 개발 라이브러리 : React
- 배포 환경 : Amazon S3, CloudFront, Github Action
- 협업 툴 : Git / Notion
